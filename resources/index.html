    # Game development resource

    Reference Website 不会画画怎么做游戏

    我经常购买的网站有 itch.io, graphicriver, gamedevmarket, unity asset store。

    而免费的可以看下：kenny 和 opengameart。

    而如果是兼职平台可以考虑下：upwork, fiverr 和 米画师

    也许有人会觉得这样没艺术性不唯一而且限制多多，我更多是觉得游戏就还没做几个先不要谈艺术性，重要的是把游戏做出来。
