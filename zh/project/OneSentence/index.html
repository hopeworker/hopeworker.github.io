        # 缘起

        自己就曾经是一个需要通过英语考试的人，后来有幸加入到一个英语学习微信群里，在那里大家互帮互助，学习到了很多的东西，最后也终于考到了理想的成绩。

        回顾当时学习英语的过程，主要碰到了两个困难。 一是没有一个系统学习的方法，另外一个是没有一个互帮互助，持之以恒，相互练习的群体。

        第一个问题，我后来找到了赖世雄老师的《英语学习方法论》。 第二个问题，就是加入到了这个英语学习群当中。后来在这个群里，有一段时间我每天坚持练习“顾家北公众号”中的“每日一句”，同时其他小伙伴也会跟着一起练习。

        有时候，一个好的习惯，往往就是从每天做一点点坚持，慢慢养成的。再加上有一起奋斗的伙伴，做起来也不会感觉辛苦。

        于是，我就在想，能否从“每日一句”做为一个切入点，将当时自己遇到的两个困难都解决呢？

        • “每日一句”，融合赖世雄的方法论，每天练习这句话的发音，词汇，词组，语法。以及尝试用自己的话来进行深入解读。

        • “每日一句”，你可以给其他人的翻译点赞，评论，甚至可以一起进行一些小游戏，比如：翻译pk，一起种树等。这样，就有了一个找共同学习伙伴的契机。
