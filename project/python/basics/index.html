# Python Basics

# Introduction

# Self introduction

大家好，我叫小熊。在 IT 行业工作了 10 多年，使用 python 也有很多年了。 在这一些的课程中，我将和大家介绍一些 Python 的基本概念和分享一些自己的编程经验，希望能和大家共同进步。

# What's programming

首先，我想先聊一聊什么的是编程。 如果把电脑比喻为一个人的话，那么编程就是和这个人交流的一种方式。 就像我们平常和别人交流，需要用到语言。在中国我们用中文，在外国我们用英文。 同样和计算机交流，就要使用编程语言。java，c，python 等等，就是我们和电脑交流的语言。 编程就是使用这些语言和电脑进行交流的一个活动。

在现实生活中，我们使用语言来 传达信息，组织活动，交流情感等等。 在电脑世界里，由于目前的阶段电脑还不够智能，我们基本上主要是使用编程语言来给计算机分配任务。 我们制定任务的步骤，电脑只能一步一步执行。 现在人工智能发展很快，电脑已经有一些自我学习的能力了。但是大部分情况电脑还是按照我们预设的步骤执行的。 也许在将来，我们可以直接使用人类的语言来和电脑交流。那时候，大家可能就不需要学习 java，c，Python 这些了。

# Why Python

接下来，我想聊一聊 Python。 编程语言其实有很多种。目前世界上大概有 700 多种编程语言，常用的编程语言有 50 多种。 大家可以上 tiobe 上看看编程语言的排行榜: https://www.tiobe.com/tiobe-index/

编程语言的分类方法很多种。比较熟悉的分类是 编译型和解释性。 编译型就是 代码编写完成后，需要编译为一个二进制的文件，才可以让 cpu 来运行。 c，c++就是编译型的语言。 解释性 则是在计算机中多加了一个虚拟运行环境，代码编写完成后，由这个虚拟环境来解释执行。 java，Python 就是解释性的语言。

我自己感觉，编程语言的差别和洗碗的差别有点像。 有的编程语言只给了你提供了水和抹布和脸盆，需要你自己把碗筷擦干净，并且把盆里的污水倒掉。 比如 C 语言就是这样。

而有的编程语言提供了你水，抹布和洗碗池，你把碗筷擦干净后，污水会自动从洗碗池里留走。 比如 Java，Python 语言就是这样。

但是自动化程度和可定制性往往是一对矛盾。 自动化程度高的的东西用起来方便，但是需要对细节修改的时候就会比较麻烦。 而如果对每个细节都可以操控的话，为了完成一个功能需要花费的工序就会比较多。 因此，每个编程语言都有他的优缺点。没有谁好谁坏之分。 需要根据不同的情况，选择适合的。

Python 语言的优点是，可以比较流畅自然的表达 程序员的想法。入门简单，使用领域广泛。 缺点是内部机制复杂，深入学习并全面掌握，需要花费比较长的时间。 同时对于多线程的支持还不是很好。

目前 Python 主要应用的领域有，人工智能，数据分析，自动化运维，自动化测试，办公自动化，web 开发，网络爬虫，脚本编写等。
